## 📚 项目概述

本仓库是《高等数学基础》（解玖霞，东南大学出版社）教材的配套实践资源库。我们通过**数据科学工具**（Python + Jupyter Notebook）重构传统数学教学，致力于实现“理论 → 实践 → 应用”的闭环学习。

**核心特色：**
* **工具驱动**：使用 `numpy`、`matplotlib`、`sympy`、`scipy` 等前沿库实现核心数学概念的落地。
* **场景化学习**：精心设计 5 个真实工业/商业场景项目，覆盖资源配置、数据拟合、周期建模和市场预测。
* **可视化呈现**：所有案例均包含直观图表（集合文氏图、函数曲线、优化区域可行域着色等）。
* **分层设计**：遵循“基础知识 → 综合项目”的循序渐进模式，适合不同阶段的学习者。

## 🚀 5大实战项目

### 1️⃣ 成绩分析与集合运算
* **应用场景**：班级三科成绩统计（数学、英语、语文）。
* **核心技术**：集合交、并、补运算 + 三维文氏图（`matplotlib_venn`）。
* **核心产出**：直观可视化展示“三门全过、至少一门、两门组合、全挂科”的学生群体分布。
* **📎 对应文件**：[专业数学 - 集合.ipynb](./专业数学-集合.ipynb)

### 2️⃣ 线性规划 - 最优资源配置
* **应用场景**：工厂生产 A、B 两款产品，受限于机器工时与人工时间，求解最大利润配置方案。
* **核心技术**：不等式约束区域绘制 + 目标函数等值线分析 + 整数解枚举。
* **核心产出**：可行域填充图（包含所有整数解标记点）、最优解精确坐标及最大总利润值。
* **📎 对应文件**：[专业数学 - 不等式.ipynb](./专业数学-不等式.ipynb)

**数学模型**：

* **约束条件**：

$$
\begin{aligned}
& 2x + y \leq 100 && \text{(机器时间限制)} \\
& x + 2y \leq 80 && \text{(人工时间限制)} \\
& x \geq 0; y \geq 0 && \text{(非负约束)}
\end{aligned}
$$

* **目标函数**：

$$
max Z = 40x + 30y \quad \text{(最大化利润)}
$$

### 3️⃣ 指数衰减拟合 - 电池容量预测
* **应用场景**：对电池使用时间与剩余容量之间的关系进行数学建模。
* **核心技术**：`curve_fit` 指数函数拟合 + $R^2$ 决定系数评估 + 外推预测。
* **核心产出**：精确参数化拟合函数、参数物理含义解释（初始容量/衰减率/标准误差）、10-20小时容量预测曲线。
* **📎 对应文件**：[专业数学 - 函数.ipynb](./专业数学-函数.ipynb)

**数学公式**：

$$
y = 99.77 \times e^{-0.099x}
$$

* **关键代码片段**：
  ```python
  def exp_func(x, a, b):
      return a * np.exp(b * x)
      
  # 执行模型拟合与评估
  params, cov = curve_fit(exp_func, x_data, y_data)
  r2 = r2_score(y_data, exp_func(x_data, *params))
  ```

### 4️⃣ 三角函数建模 - 昼长变化预测
* **应用场景**：模拟并预测特定地区一年 12 个月的昼长周期性变化现象。
* **核心技术**：余弦函数 $A \cdot \cos(Bx + C) + D$ 曲线拟合 + 跨年长效预测。
* **核心产出**：拟合参数物理含义解析、未来 12 个月昼长精确预测值与变化趋势图。
* **📎 对应文件**：[专业数学 - 函数.ipynb](./专业数学-函数.ipynb)

**数学公式**：

$$
y = -2.63 \times \cos(0.91x + 0.76) + 11.75
$$

> **参数物理含义解析**：
> * $A$ ：代表振幅（白昼变动范围）
> * $B$ ：决定年周期
> * $C$ ：代表相位偏移（决定最长白昼出现的月份）
> * $D$ ：代表平均昼长

---

### 5️⃣ 复合模型拟合 - 智能手机市场预测
* **应用场景**：基于 1980-2024 年全球智能手机销量历史数据，预测 2025-2040 年的长远趋势。
* **核心技术**：8 种不同数学模型的横向对比 + MSE（均方误差）评估机制 + 15年长线外推。
* **📎 对应文件**：[专业数学 - 函数.ipynb](./专业数学-函数.ipynb)

**模型库示例**：

* **单指数模型**：

$$
y = a 	\times e^{bx}
$$

* **双指数 + 幂函数复合模型**：

$$
y = a 	\times e^{bx} + c 	\times e^{dx} + e 	\times x^f
$$

* **双指数 + 对数复合模型**：

$$
y = a 	\times e^{bx} + c 	\times e^{dx} + e 	\times \ln(x)
$$

## 🎓 教学特色

### 对比传统教学的优势

| 维度 | 传统教学 | 本项目方法 |
| :--- | :--- | :--- |
| **集合运算** | 抽象概念，仅能依靠手工绘制简单的二维文氏图 | 通过 Python 自动生成高维、精确且具备交互性的文氏图 |
| **不等式求解** | 繁琐的手工代数计算，配以粗略的数轴草图 | 利用强大的符号计算库，一键获取精确解集并完美可视化 |
| **函数拟合** | 复杂的数学公式推导，难以处理非线性的真实数据 | 结合真实工业数据，进行多模型横向对比与 $R^2$ 科学评估 |
| **线性规划** | 局限于平面图解法，难以处理多约束与整数解问题 | 约束区域自动化着色，支持全空间内的整数解智能枚举 |

### 培养的核心能力
* **数学建模能力**：培养将复杂的实际业务场景转化为抽象数学模型的核心素养。
* **编程实现能力**：熟练使用 Python 前沿科学计算工具库高效执行数学算法。
* **数据分析能力**：掌握数据拟合、外推预测、模型评估以及优劣筛选的标准流程。
* **可视化表达**：学会利用专业、直观的图表清晰地传达数学结论。
