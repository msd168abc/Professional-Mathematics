**核心特色:**
- ✅ **工具驱动**: 使用`numpy`/`matplotlib`/`sympy`/`scipy`等库实现数学概念
- ✅ **场景化学习**: 5个真实项目覆盖资源配置/数据拟合/周期建模/市场预测
- ✅ **可视化呈现**: 所有案例包含交互式图表(集合文氏图/函数曲线/优化区域)
- ✅ **分层设计**: 基础知识→综合项目,适合不同学习阶段
