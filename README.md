🚀 5 大实战项目

1️⃣ 成绩分析与集合运算

场景: 班级三科成绩统计（数学 / 英语 / 语文）
技术: 集合交并补运算 + 三维文氏图 (matplotlib_venn)
产出: 可视化展示“三门全过 / 至少一门 / 两门组合 / 全挂科”学生分布

2️⃣ 线性规划 - 最优资源配置

场景: 工厂生产 A/B 两产品，受机器时间/人工限制，求最大利润配置
技术: 不等式约束区域 + 目标函数等值线 + 整数解枚举
产出: 可行域填充图（含整数解标记点）+ 最优解坐标 + 最大利润值

数学模型：

begin{cases}
text{约束条件:} & 2x + y leq 100 quad (text{机器时间}) \
& x + 2y leq 80 quad;; (text{人工时间}) \
& x, y geq 0 \
text{目标函数:} & Z = 40x + 30y quad (text{最大化利润})
end{cases}

3️⃣ 指数衰减拟合 - 电池容量预测

场景: 电池使用时间与剩余容量关系建模
技术: curve_fit 指数函数拟合 + R² 评估 + 外推预测
产出: 拟合函数、参数含义解释（初始容量/衰减率/标准误差）、10-20h 容量预测曲线

关键代码片段：

python
def exp_func(x, a, b):
    return a * np.exp(b * x)

params, cov = curve_fit(exp_func, x_data, y_data)
r2 = r2_score(y_data, exp_func(x_data, *params))
拟合结果: y = 99.77 * exp(-0.099x)

4️⃣ 三角函数建模 - 昼长变化预测

场景: 模拟某地一年 12 个月昼长变化（周期现象）
技术: 余弦函数 Acos(Bx+C)+D 拟合 + 跨年预测
产出: 拟合参数含义（振幅/周期因子/相位偏移/平均昼长）、未来 12 个月昼长预测值

数学公式：

y = -2.63 cdot cos(0.91x + 0.76) + 11.75

A = 振幅（白昼变动范围）
B = 1 表示年周期
C = 相位偏移（决定最长白昼出现月份）

5️⃣ 复合模型拟合 - 智能手机市场预测

场景: 1980-2024 年市场销量数据，预测 2025-2040 趋势
技术: 8 种模型对比 + MSE 误差评估选择最优模型 + 15 年外推预测

模型库示例：
模型类型   表达式
单指数   a cdot e^{bx}

双指数 + 幂函数   a cdot e^{bx} + c cdot e^{dx} + e cdot x^f

双指数 + 对数   a cdot e^{bx} + c cdot e^{dx} + e cdot ln(x)
